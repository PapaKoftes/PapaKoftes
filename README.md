# Hi there 👋

**Mina Mikail** — Mechatronics engineer focused on **CNC automation**, with a long-term interest in **robotics** and **home automation**. Self-taught in software; I use AI-assisted coding as a learning tool while building production tooling.

**Highlights:** **ezdxf** — layered DXF (validation, generation, nesting). production DXF validation and ISO generation for Biesse Machines; **Random Forest** milling-time predictor (42-feature model from actual run data). **OPC UA** Windows service for Biesse machine data (cert auth, JSON export for monitoring/invoicing). **Airtable/Excel** — invoicing pipeline (fetch, extract, verify, reconcile, report); consolidated from 118+ scripts into 8 modules.
